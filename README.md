Just a calendar*

## How to run
- Install the dependencies
```shell
npm install
```

- Build with Grunt
```shell
grunt
```

- Serve the `dist` folder (example using node module http-server)
```shell
http-server
```

## Running the tests
```shell
npm test
```

## Technologies
- Node.js
- Grunt
- Jest
- JavaScript ECMA-262 2015 (a.k.a ES6)
- 6to5 (now Babel) transpiler
- Browserify
